## Running Local

Install requirements: 

```bash
sudo gem install  -n /usr/local/bin bundler jekyll

#bundle init

bundle install --path vendor/bundle
```

Execute in Local:

```bash
bundle exec jekyll serve
````


## Deploy to Production

just push in master
